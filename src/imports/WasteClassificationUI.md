Diseña la interfaz de una aplicación de escritorio para un sistema de clasificación automática de residuos reciclables mediante visión por computadora.

El sistema está desarrollado en Python y funciona en CPU. Es una aplicación técnica con enfoque académico (Ingeniería Informática), no comercial.

Estilo visual:
- Minimalista
- Profesional
- Fondo claro
- Colores neutros (gris, blanco) con acentos verdes (sostenibilidad)
- Diseño limpio tipo dashboard técnico
- Tipografía moderna sans-serif

Pantallas requeridas:

1. Pantalla Principal (Dashboard)
- Título: “Sistema de Clasificación de Residuos”
- Vista previa grande de cámara o imagen cargada
- Botón “Capturar imagen”
- Botón “Cargar imagen”
- Botón “Clasificar”
- Panel lateral derecho con:
  - Resultado de clasificación
  - Nivel de confianza (%)
  - Estado del modelo (Cargado / No cargado)

2. Pantalla de Resultado
- Imagen analizada en grande
- Etiqueta destacada con categoría detectada (ej. “Plástico”)
- Barra de progreso mostrando confianza
- Botón “Nueva clasificación”
- Historial pequeño de clasificaciones recientes

3. Pantalla de Gestión de Categorías (Administrador)
- Lista de categorías existentes (plástico, vidrio, papel, metal)
- Botón “Agregar categoría”
- Botón “Editar”
- Botón “Eliminar”
- Diseño tipo tabla simple

4. Pantalla de Entrenamiento del Modelo
- Botón “Iniciar entrenamiento”
- Indicador de progreso
- Gráfica simple de precisión vs épocas
- Métricas mostradas en tarjetas:
  - Accuracy
  - Loss
  - Número de imágenes del dataset

Diseño técnico:
- Layout basado en grid
- Componentes reutilizables
- Botones con estados (hover, activo, deshabilitado)
- Indicadores de estado (verde = correcto, rojo = error)

El diseño debe parecer una herramienta técnica universitaria funcional, no una app móvil ni una plataforma comercial.
